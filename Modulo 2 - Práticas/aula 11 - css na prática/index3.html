Conceitos Básicos de CSS

Página Inicial

Os códigos HTML permitem que eu crie três tipos de listas. A lista não-ordenada, que recebe um marcador em cada item. A lista ordenada, que recebe uma numeração em cada item. E, por fim, a lista descritiva que serve para criar itens e suas descrições.

Treinando os seletores

Também é possivel criar tabelas utilizando códigos HTML. As tabelas são elementos fundamentalmente estruturados por meio de colunas, linhas e células. As colunas são os elementos que ficam na posição vertical, as linhas na posição horizontal e as células são as intersecções.
